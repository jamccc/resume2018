# PC端简历网站

该项目是一个PC端简历网站。[预览链接](https://jamloveu.github.io/resume2018/resume2018.html)

## 功能介绍：

1. 进入网站加载动画
2. 顶部导航 + 自动高亮 + 二级菜单
3. 内容模块自动向上滚动
4. 作品集的轮播
5. 留言功能

## 实现细节

使用了 HTML5 + CSS3 + 原生JavaScript + Swiper轮播库，使用 AJAX + LeanCloud将用户留言保存到服务器并读取展示。

## 技术栈

HTML5 | CSS3 | 原生JavaScript | AJAX | Swiper轮播库 | CSS插件
